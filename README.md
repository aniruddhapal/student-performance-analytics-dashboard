# 🎯 Student Performance Analytics Dashboard (Power BI)

## 📌 Problem
Schools track marks but often miss performance trends and at-risk students, making timely intervention difficult.

---

## 💡 Solution
This Power BI dashboard identifies:

- Students at risk (≤40% performance)
- Students showing decline across terms
- Class-wise risk distribution
- Clear intervention list for teachers

---

## 🧠 Key Features

- 📊 Class-wise risk analysis (% declining students)
- 🚨 High-risk student identification (≤40%)
- 📉 Trend tracking (T1 vs T2 performance)
- ❌ Separation of absentee vs actual decline
- 🎯 Actionable student-level intervention list

## ⚙️ Approach

- Data modeling using **Fact & Dimension tables**
- Student-level aggregation (T1 vs T2)
- Attendance-aware filtering
- DAX measures for KPIs and risk metrics

---

## 📊 Key Insights

- ~X% of students are declining in certain classes
- Only X students fall below 40%, but X% lie in the 40–60% risk band
- Specific classes show higher risk despite similar averages

---

## 🛠 Tools Used

- Power BI  
- DAX  
- Python (data preprocessing)

## 🚀 Outcome

A decision-focused dashboard that helps schools:
- Identify at-risk students quickly
- Prioritize intervention
- Improve academic outcomes
